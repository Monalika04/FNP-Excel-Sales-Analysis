# Ferns and Petals Sales Analysis Dashboard

## Overview
Built an interactive Excel dashboard to analyze sales performance for Ferns and Petals (FNP).  
The dashboard provides business insights into revenue trends, customer behavior, product performance, and delivery efficiency.

---

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning

## Key Metrics
- Total Orders: 1000
- Total Revenue: ₹3.52M
- Average Customer Spend: ₹3520.98
- Average Delivery Time: 5.53 Days

---

## Key Insights
- Anniversary generated the highest revenue
- Colors category contributed maximum sales
- Sales peaked during festive seasons
- Top cities contributed most orders
